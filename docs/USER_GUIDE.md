## Getting Started

### First Launch

When you first launch MyDeckCards, you'll see an empty home screen with a floating "+" button. This is where all your flashcard decks will be displayed.

### Creating Your First Deck

1. Tap the **"+"** button in the bottom right corner
2. Enter a name for your deck (e.g., "Spanish Vocabulary", "Geography Quiz")
3. (Optional) Tap the emoji icon to select a custom icon for your deck
4. Tap **"Create"** / **"Crear"**

Your deck will appear on the home screen, showing 0 cards initially.

## Deck Management

### Viewing Decks

Decks are displayed in either grid or list view. Each deck shows:
- Deck name
- Custom emoji icon (or default üìö)
- Number of cards in the deck
- Quick action buttons

### Sorting Decks

Tap the sort button (‚ãÆ) in the app bar to organize your decks:
- **Name (A-Z)**: Alphabetical order
- **Name (Z-A)**: Reverse alphabetical order
- **Newest First**: Most recently created decks first
- **Oldest First**: Oldest decks first
- **Most Cards**: Decks with the most cards first
- **Least Cards**: Decks with the fewest cards first

Your sort preference is saved and persists between app launches.

### Renaming a Deck

1. Long press on the deck you want to rename
2. Select **"Rename"** / **"Renombrar"**
3. Enter the new name
4. Tap **"Rename"** / **"Renombrar"** to confirm

### Changing Deck Icon

1. Long press on the deck
2. Select **"Change Icon"** / **"Cambiar Icono"**
3. Browse through available emoji categories:
   - Smileys & People
   - Animals & Nature
   - Food & Drink
   - Travel & Places
   - Activities
   - Objects
   - Symbols
   - Flags
4. Tap an emoji to select it

### Archiving a Deck

Instead of deleting decks, you can archive them:

1. Long press on the deck you want to archive
2. Select **"Archive"** / **"Archivar"**
3. The deck moves to the archived section

Archived decks don't appear in the main view but can be restored at any time.

### Deleting a Deck

To permanently delete a deck:

1. First archive the deck (see above)
2. Open the drawer menu (‚ò∞)
3. Select **"Archived Decks"** / **"Mazos Archivados"**
4. Long press the deck you want to delete
5. Select **"Delete Permanently"** / **"Eliminar Permanentemente"**
6. Confirm the action

**Warning**: Permanent deletion cannot be undone. Consider exporting the deck first.

## Card Management

### Adding Cards

1. Tap on a deck to open it
2. Tap the **"+"** button in the bottom right corner
3. Enter the **front** of the card (question, word, term)
4. Enter the **back** of the card (answer, definition, translation)
5. Tap **"Add"** / **"Agregar"**

The dialog will remain open, allowing you to quickly add multiple cards. Tap **"Cancel"** / **"Cancelar"** when done.

### Editing Cards

**Method 1: Tap to Edit**
1. Open a deck
2. Tap directly on any card
3. Modify the front or back text
4. Tap **"Save"** / **"Guardar"**

**Method 2: Long Press Menu**
1. Long press on a card
2. Select **"Edit"** / **"Editar"**
3. Make your changes
4. Tap **"Save"** / **"Guardar"**

### Deleting Cards

1. Long press on the card you want to delete
2. Select **"Delete"** / **"Eliminar"**
3. Confirm the deletion

## Review Mode

Review mode is where you test your knowledge by going through flashcards.

### Starting a Review

1. From the home screen, tap on a deck to open it
2. Tap the **play icon** (‚ñ∂) in the top right corner
3. The review session begins with the first card

### During Review

**Viewing the Answer:**
- The card shows the front (question) initially
- Tap the card once to flip it and reveal the answer
- Once flipped, the card cannot be flipped back (this prevents cheating!)

**Answering:**

You have two ways to indicate if you knew the answer:

1. **Swipe Gestures:**
   - Swipe **RIGHT** (‚Üí) if you knew the answer
   - Swipe **LEFT** (‚Üê) if you didn't know it

2. **Buttons:**
   - Tap the **‚úì** (checkmark) button if you knew it
   - Tap the **‚úó** (X) button if you didn't

**Progress Tracking:**
- The top of the screen shows your position (e.g., "Card 3 / 10")
- Below that, you'll see your score:
  - Green number: Correct answers
  - Red number: Incorrect answers

### Ending a Review

**Completing All Cards:**
When you reach the last card, you'll see a final summary:
- Total cards reviewed
- Number of correct answers
- Number of incorrect answers
- Percentage score

You can then:
- **Review Again**: Start over with the same deck
- **Exit**: Return to the deck detail screen

**Exiting Early:**
If you try to exit before completing all cards:
1. A confirmation dialog appears
2. Choose to **Continue** reviewing or **Exit**
3. Your progress is not saved if you exit early

### Review Features

- **Screen stays awake**: On mobile devices, the screen won't turn off during review
- **Randomization**: Cards are not randomized by default (shown in order)
- **Single flip**: Each card can only be flipped once per review to maintain integrity

## Archived Decks

2. Select **"Archived Decks"** / **"Mazos Archivados"**

## Import/Export

Export and import decks to backup your data or share with others.

### Exporting Decks

1. Open the drawer menu (‚ò∞)
2. Select **"Export Decks"** / **"Exportar Mazos"**
3. Read the information dialog:
   - Shows how many decks will be exported
   - Explains that all decks (including archived) are exported
4. Tap **"Export"** / **"Exportar"**
5. Choose a folder to save the file
6. The file is named automatically: `flashcards_export_YYYYMMDD_HHMMSS.json`

### Importing Decks

1. Open the drawer menu (‚ò∞)
2. Select **"Import Decks"** / **"Importar Mazos"**
3. Read the information dialog
4. Tap **"Select File"** / **"Seleccionar Archivo"**
5. Choose a `.json` file to import
6. Review the import summary:
   - Number of decks to be imported
   - Total number of cards
7. Tap **"Import"** / **"Importar"**
8. A success message shows how many decks were imported

### Import Behavior

- **Duplicate names**: If an imported deck has the same name as an existing deck, it's automatically renamed (e.g., "Deck Name (1)")
- **IDs**: New unique IDs are generated for all decks and cards
- **Archived status**: The archived status is preserved during import
- **Icons**: Custom icons are preserved if present in the JSON

### JSON File Format

You can create or edit JSON files manually. Format:

```json
{
  "version": "1.0",
  "exportDate": "2026-01-15T10:30:00.000Z",
  "decks": [
    {
      "name": "My Deck",
      "icon": "üìö",
      "archived": false,
      "cards": [
        {
          "front": "Question",
          "back": "Answer"
        }
      ]
    }
  ]
}
```

## Tips & Tricks

### Efficient Card Creation

- Use the continuous add dialog to quickly create multiple cards
- Copy and paste from other sources (spreadsheets, documents)
- Consider using the import feature for bulk card creation

### Organizing Large Collections

- Use descriptive deck names (e.g., "Spanish Verbs - Present Tense")
- Assign emoji icons to categories (e.g., üá™üá∏ for Spanish, üî¨ for Science)
- Use sorting to find decks quickly
- Archive old or seasonal decks to reduce clutter

### Effective Review Strategy

1. **Spaced Repetition**: Review decks regularly, not just once
2. **Honest Self-Assessment**: Be truthful when marking cards right/wrong
3. **Focus Sessions**: Review one deck at a time for better retention
4. **Regular Practice**: Short, daily sessions are more effective than long, infrequent ones

### Backup Best Practices

- Export your decks regularly (weekly or monthly)
- Store backups in multiple locations (cloud storage, external drive)
- Before major changes, export your current state
- Test imports occasionally to ensure backups work

### Performance Tips

- Keep individual decks under 500 cards for optimal performance
- Delete unused cards rather than keeping empty decks
- Close and reopen the app if it becomes slow

### Sharing Decks

1. Export the deck you want to share
2. Send the JSON file via email, messaging, or cloud storage
3. The recipient imports it using the import feature
4. They can rename or modify the deck as needed

### Creating Custom Predefined Decks

1. Create and populate a deck with cards
2. Export it to get the JSON file
3. Share with others or keep as a template
4. Import when needed to create a new copy

## Troubleshooting

### Cards Not Showing

- Ensure you've added cards to the deck
- Check if the deck is archived (it won't appear in main view)
- Try sorting by different criteria

### Import Failed

- Verify the file is valid JSON
- Check that the file has a `.json` extension
- Ensure the JSON structure matches the required format
- Look for missing required fields (name, cards)

### Review Not Working

- Ensure the deck has at least one card
- Try closing and reopening the deck
- Check if the app has necessary permissions (screen wake lock on mobile)

### Theme Not Changing

- If using "System" theme, check your device's theme settings
- Try manually selecting Light or Dark
- Restart the app if the theme doesn't update

### Lost Data

- Check the archived decks view
- Look for export files in your usual save location
- If using system theme, data is stored locally and persists across sessions
